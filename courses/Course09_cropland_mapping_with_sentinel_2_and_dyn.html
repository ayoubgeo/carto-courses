Sentinel-2 Data Acquisition and Cloud Masking
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Sentinel-2 Data Acquisition and Cloud Masking
// Define region of interest (Rif Mountains and Middle Atlas, Morocco)
var roi = ee.Geometry.Rectangle([-6, 32, -2, 35]);

// Function to mask clouds using the QA60 band
var maskS2clouds = function(image) {
  var qa = image.select('QA60');
  // Bits 10 and 11 are clouds and cirrus, respectively.
  var cloudBitMask = 1 << 10;
  var cirrusBitMask = 1 << 11;
  // Both flags should be set to zero, indicating clear conditions.
  var mask = qa.bitwiseAnd(cloudBitMask).eq(0)
      .and(qa.bitwiseAnd(cirrusBitMask).eq(0));
  return image.updateMask(mask).divide(10000);
};

// Load Sentinel-2 SR data and filter by date and region
var s2 = ee.ImageCollection('COPERNICUS/S2_SR')
    .filterDate('2023-01-01', '2023-12-31')
    .filterBounds(roi)
    .map(maskS2clouds);
🔑
Beginner note: This code block first defines the region of interest (ROI) in North Morocco. Then, it defines a function `maskS2clouds` to mask out clouds from Sentinel-2 images using the QA60 band. Finally, it loads the Sentinel-2 Surface Reflectance dataset, filters it by date and region, and applies the cloud masking function to each image in the collection. The values are divided by 10000 to convert to reflectance values.

Vegetation Index Calculation and Dynamic World Integration
// Function to calculate NDVI
var addNDVI = function(image) {
  var ndvi = image.normalizedDifference(['B8', 'B4']).rename('NDVI');
  return image.addBands(ndvi);
};

// Function to calculate EVI
var addEVI = function(image) {
  var evi = image.expression(
      '2.5 * (NIR - RED) / (NIR + 6 * RED - 7.5 * BLUE + 1)', {
        'NIR': image.select('B8'),
        'RED': image.select('B4'),
        'BLUE': image.select('B2')
      }).rename('EVI');
  return image.addBands(evi);
};

// Apply NDVI and EVI calculations
var s2_ndvi_evi = s2.map(addNDVI).map(addEVI);

// Load Dynamic World data
var dw = ee.ImageCollection('GOOGLE/DYNAMIC_WORLD/V1')
    .filterDate('2023-01-01', '2023-12-31')
    .filterBounds(roi);

// Get the most frequent land cover class from Dynamic World
var dw_mode = dw.select('label').reduce(ee.Reducer.mode());
🔑
Beginner note: This code block defines functions to calculate NDVI and EVI from Sentinel-2 imagery. It then applies these functions to the Sentinel-2 image collection. It also loads the Dynamic World dataset, filters it by date and region, and calculates the most frequent land cover class for each pixel using the `reduce(ee.Reducer.mode())` function.

Export to Google Drive
// Export the cropland map to Google Drive
Export.image.toDrive({
  image: dw_mode,
  description: 'Cropland_Map_Morocco_2023',
  folder: 'GEE_Exports',
  scale: 10,
  region: roi,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
