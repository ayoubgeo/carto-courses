Setting up the Environment and Filtering Data
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Setting up the Environment and Filtering Data
// Define the region of interest (Rif Mountains and Middle Atlas)
var roi = ee.Geometry.Polygon([
    [-5.5, 35.0], [-3.5, 35.0], [-3.5, 33.0], [-5.5, 33.0]
]);

// Define the date range
var startDate = '2022-01-01';
var endDate = '2022-12-31';

// Load Landsat 8 surface reflectance data
var landsat8 = ee.ImageCollection('LANDSAT/LC08/C01/T1_SR')
    .filterDate(startDate, endDate)
    .filterBounds(roi);
🔑
Beginner note: This code block initializes the analysis by defining the region of interest (ROI) as a polygon covering the Rif Mountains and Middle Atlas. It then sets the start and end dates for the analysis period. Finally, it loads the Landsat 8 surface reflectance image collection from Google Earth Engine and filters it by date and region.

Cloud Masking and NDVI Calculation
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

Cloud Masking and NDVI Calculation
// Function to mask clouds using the pixel_qa band
var maskClouds = function(image) {
  var qa = image.select('pixel_qa');
  var cloudBitMask = 1 << 5;
  var cloudShadowBitMask = 1 << 3;
  var mask = qa.bitwiseAnd(cloudBitMask).eq(0)
      .and(qa.bitwiseAnd(cloudShadowBitMask).eq(0));
  return image.updateMask(mask);
};

// Apply cloud masking
var landsat8_masked = landsat8.map(maskClouds);

// Function to calculate NDVI
var calculateNDVI = function(image) {
  return image.normalizedDifference(['B5', 'B4']).rename('NDVI');
};

// Calculate NDVI for each image
var ndviCollection = landsat8_masked.map(calculateNDVI);
🔑
Beginner note: This code block defines a function to mask clouds in the Landsat 8 imagery using the pixel_qa band. It then applies this cloud masking function to the entire image collection. After cloud masking, it defines another function to calculate NDVI using the near-infrared (B5) and red (B4) bands. Finally, it applies the NDVI calculation function to the cloud-masked image collection.

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Composite the NDVI collection and export the mean NDVI image.
var ndvi_composite = ndviCollection.median();

Export.image.toDrive({
  image: ndvi_composite,
  description: 'NDVI_North_Morocco_2022',
  folder: 'GEE_Exports',
  region: roi,
  scale: 30,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
