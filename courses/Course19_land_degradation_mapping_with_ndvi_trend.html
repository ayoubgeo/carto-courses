Data Acquisition and NDVI Calculation
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Data Acquisition and NDVI Calculation
// Define region of interest (North Morocco).
var geometry = ee.Geometry.Rectangle([-6, 32, -2, 35]);

// Function to calculate NDVI.
var addNDVI = function(image) {
  var ndvi = image.normalizedDifference(['B5', 'B4']).rename('NDVI');
  return image.addBands(ndvi);
};

// Load Landsat 8 data and filter by date and location.
var collection = ee.ImageCollection('LANDSAT/LC08/C01/T1_SR')
    .filterDate('2015-01-01', '2023-12-31')
    .filterBounds(geometry)
    .map(addNDVI);

// Function to mask clouds using the pixel_qa band.
var maskClouds = function(image) {
  var qa = image.select('pixel_qa');
  var cloudShadowBitMask = 1 << 3;
  var cloudsBitMask = 1 << 5;
  var mask = qa.bitwiseAnd(cloudShadowBitMask).eq(0)
      .and(qa.bitwiseAnd(cloudsBitMask).eq(0));
  return image.updateMask(mask);
};

collection = collection.map(maskClouds);
🔑
Beginner note: This code block first defines the region of interest and then calculates the NDVI for each Landsat 8 image. It loads the Landsat 8 Surface Reflectance dataset, filters it by date and location, and applies a cloud masking function to improve data quality. The addNDVI function calculates NDVI using the red (B4) and near-infrared (B5) bands. The maskClouds function uses the pixel_qa band to mask out cloudy pixels.

NDVI Trend Analysis
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

NDVI Trend Analysis
// Create a time series of NDVI.
var ndviTimeSeries = collection.select('NDVI').toBands();

// Calculate a linear trend.
var trend = collection.select(['NDVI', 'system:time_start'])
  .reduce(ee.Reducer.linearFit());

// Get the slope of the trend (NDVI change per unit of time).
var slope = trend.select('scale').rename('NDVI_trend');

// Get the p-value of the trend.
var pValue = trend.select('p');

// Mask out insignificant trends (p > 0.05).
var significantTrend = slope.updateMask(pValue.lte(0.05));

// Display the trend.
Map.addLayer(significantTrend, {min: -0.1, max: 0.1, palette: 'red, white, green'}, 'NDVI Trend');
🔑
Beginner note: This code block performs a linear regression analysis on the NDVI time series to calculate the trend. The `ee.Reducer.linearFit()` function calculates the slope and p-value of the trend. The slope represents the NDVI change over time. The p-value indicates the statistical significance of the trend. The code then masks out insignificant trends (p > 0.05) and displays the significant trends on the map, with red indicating decreasing NDVI (degradation) and green indicating increasing NDVI (improvement).

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Export the NDVI trend image to Google Drive.
Export.image.toDrive({
  image: slope,
  description: 'NDVI_trend_North_Morocco',
  folder: 'GEE_Exports',
  scale: 30,
  region: geometry,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
