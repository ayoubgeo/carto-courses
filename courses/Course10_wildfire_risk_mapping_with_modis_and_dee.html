Import and Filter MODIS Fire Data
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Import and Filter MODIS Fire Data
// Define the region of interest (Rif Mountains and Middle Atlas, Morocco)
var roi = ee.Geometry.Rectangle([-6, 32, -2, 35]);

// Filter MODIS active fire data for 2023
var fires = ee.ImageCollection('MODIS/061/MCD14ML/C6_1')
  .filterDate('2023-01-01', '2023-12-31')
  .filterBounds(roi);

// Function to mask out pixels with low confidence
var maskLowConfidence = function(image) {
  return image.updateMask(image.select('confidence').gt(50));
};

// Apply the mask
vAR firesFiltered = fires.map(maskLowConfidence);

//Mosaic the fire collection and select the FRP band
var fire_frp = firesFiltered.mosaic().select('frp');
🔑
Beginner note: This code block imports the MODIS active fire data, filters it for the year 2023 and the defined region of interest (North Morocco). A function is defined to mask out low confidence fire detections. Finally the fire collection is mosaicked and the Fire Radiative Power (FRP) band is selected.

Calculate NDVI and EVI
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

Calculate NDVI and EVI
// Function to calculate NDVI
var calculateNDVI = function(image) {
  return image.normalizedDifference(['Nir', 'Red']).rename('NDVI');
};

// Function to calculate EVI
var calculateEVI = function(image) {
  return image.expression(
    '2.5 * ((NIR - RED) / (NIR + 6 * RED - 7.5 * BLUE + 1))', {
      'NIR': image.select('Nir'),
      'RED': image.select('Red'),
      'BLUE': image.select('Blue')
    }).rename('EVI');
};

// Load MODIS surface reflectance data
var modisSR = ee.ImageCollection('MODIS/061/MOD09GA/006')
  .filterDate('2023-01-01', '2023-12-31')
  .filterBounds(roi);

// Apply scaling factors and rename bands
var modisSRScaled = modisSR.map(function(image) {
  return image.multiply(0.0001).select('sur_refl_b01', 'sur_refl_b02', 'sur_refl_b04').rename('Red', 'Nir', 'Blue').copyProperties(image, ['system:time_start']);
});

// Calculate NDVI and EVI
var ndvi = modisSRScaled.map(calculateNDVI).mosaic();
var evi = modisSRScaled.map(calculateEVI).mosaic();
🔑
Beginner note: This code calculates the Normalized Difference Vegetation Index (NDVI) and Enhanced Vegetation Index (EVI) using MODIS surface reflectance data. Functions are defined for each index, and the MODIS data is filtered, scaled, renamed, and then used to calculate the indices.
