Importing and Filtering Data
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Importing and Filtering Data
// Define the region of interest (Rif Mountains and Middle Atlas, Morocco)
var roi = ee.Geometry.Rectangle([-6, 32, -2, 35]);

// Import ERA5-Land hourly data for 2022
var era5 = ee.ImageCollection('ECMWF/ERA5_LAND/HOURLY')
  .filterDate('2022-01-01', '2022-12-31')
  .filterBounds(roi);

// Print the number of images in the collection
print('Number of ERA5 images:', era5.size());

// Import SRTM DEM data
var dem = ee.Image('USGS/SRTMGL1_003').clip(roi);

// Import land cover data
var landcover = ee.Image('COPERNICUS/Landcover/100m/Proba-V-C3/Global').select('discrete_classification').clip(roi);
🔑
Beginner note: This code block imports the necessary datasets: ERA5-Land for climate data, SRTM for elevation, and COPERNICUS for land cover. It filters the ERA5-Land data for the year 2022 and clips all datasets to the defined region of interest (ROI) in North Morocco. The print statement is useful to check that data is being imported correctly. The landcover image is also clipped to the ROI and the 'discrete_classification' band is selected.

Calculating Solar Irradiance and Terrain Attributes
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

Calculating Solar Irradiance and Terrain Attributes
// Calculate daily solar irradiance (example, needs adjustment based on ERA5 bands)
var dailyIrradiance = era5.map(function(image) {
  // Assuming 'surface_solar_radiation_downwards' is the relevant band and converting from J/m² to kWh/m²
  var irradiance = image.select('surface_solar_radiation_downwards').multiply(2.777e-7); // J/m² to kWh/m²
  return irradiance.set('system:time_start', image.get('system:time_start'));
});

// Calculate annual solar irradiance
var annualIrradiance = dailyIrradiance.sum();

// Calculate slope and aspect from DEM
var slope = ee.Terrain.slope(dem);
var aspect = ee.Terrain.aspect(dem);

// Forest cover mask (assuming forest is classified as 5 or 6 in landcover)
var forestMask = landcover.eq(5).or(landcover.eq(6));
🔑
Beginner note: This block calculates the daily and annual solar irradiance from the ERA5-Land data. It also calculates the slope and aspect from the SRTM DEM. A forest mask is created based on the land cover data, assuming forest is classified as 5 or 6. The 'surface_solar_radiation_downwards' band from ERA5-Land is selected and converted to kWh/m². The slope and aspect are calculated using the `ee.Terrain` functions.

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Export the suitability map to Google Drive
Export.image.toDrive({
  image: annualIrradiance,
  description: 'Solar_Irradiance_North_Morocco',
  folder: 'GEE_Exports',
  scale: 30, // Adjust scale as needed
  region: roi,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
