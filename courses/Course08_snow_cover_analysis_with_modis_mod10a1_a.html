Importing and Filtering MODIS Data
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Importing and Filtering MODIS Data
// Define the region of interest (North Morocco).
var geometry = ee.Geometry.Rectangle([-6, 32, -2, 35]);

// Load the MODIS MOD10A1 snow cover dataset.
var snowCover = ee.ImageCollection('MODIS/061/MOD10A1')
  .filterDate('2020-01-01', '2023-12-31')
  .filterBounds(geometry);

// Filter for December, January, and February.
var winterSnow = snowCover.filter(ee.Filter.calendarRange(12, 2, 'month'));

print('MODIS Snow Cover Collection:', winterSnow);
🔑
Beginner note: This code block first defines a region of interest in North Morocco. Then, it loads the MODIS MOD10A1 dataset and filters it to the years 2020-2023 and the specified geometry. Finally, it filters the collection to only include images from December, January, and February using `ee.Filter.calendarRange`. The resulting `winterSnow` collection is printed to the console.

Calculating and Visualizing Mean Snow Cover
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

Calculating and Visualizing Mean Snow Cover
// Calculate the mean NDSI snow cover for each year.
var annualSnow = ee.List.sequence(2020, 2023).map(function(year) {
  var startDate = ee.Date.fromYMD(year, 12, 1);
  var endDate = ee.Date.fromYMD(ee.Number(year).add(1), 2, 28);
  var annualImage = winterSnow.filterDate(startDate, endDate)
    .mean()
    .clip(geometry);
  return annualImage.set('year', year);
});

annualSnow = ee.ImageCollection.fromImages(annualSnow);

// Define visualization parameters.
var snowVisParams = {
  min: 0.0,
  max: 100.0,
  palette: ['00008B', '0000FF', 'ADD8E6', 'FFFFFF']
};

// Display the annual mean NDSI snow cover as a map.
annualSnow.sort('year').iterate(function(image) {
  var year = image.get('year');
  Map.addLayer(image, snowVisParams, 'Mean Snow Cover ' + year);
});
🔑
Beginner note: This code calculates the mean NDSI snow cover for each year (2020-2023) by filtering the `winterSnow` collection for each year and calculating the mean using `.mean()`. The `annualSnow` is then converted to an image collection. A color palette is defined for visualization. Finally, the annual mean snow cover is displayed on the map for each year using `Map.addLayer()`.

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Export the mean snow cover image to Google Drive.
Export.image.toDrive({
  image: annualSnow.first(),
  description: 'Mean_Snow_Cover_2020',
  folder: 'GEE_Exports',
  scale: 500,
  region: geometry,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
