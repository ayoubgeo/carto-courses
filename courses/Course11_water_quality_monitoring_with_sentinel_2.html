Import and Filter Sentinel-2 Data
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Import and Filter Sentinel-2 Data
// Define region of interest (Rif Mountains and Middle Atlas, Morocco)
var roi = ee.Geometry.Rectangle([-5.5, 34, -3.5, 35]);

// Filter Sentinel-2 SR data for the specified period and region
var dataset = ee.ImageCollection('COPERNICUS/S2_SR')
                  .filterDate('2023-06-01', '2023-09-30')
                  .filterBounds(roi)
                  .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 20));

// Print the number of images found
print('Number of Sentinel-2 images:', dataset.size());

// Function to mask clouds using the QA60 band
var maskS2clouds = function(image) {
  var qa = image.select('QA60');
  // Bits 10 and 11 are clouds and cirrus, respectively.
  var cloudBitMask = 1 << 10;
  var cirrusBitMask = 1 << 11;
  // Both flags should be set to zero, indicating clear conditions.
  var mask = qa.bitwiseAnd(cloudBitMask).eq(0)
      .and(qa.bitwiseAnd(cirrusBitMask).eq(0));
  return image.updateMask(mask).divide(10000);
};
🔑
Beginner note: This code block first defines a region of interest (ROI) encompassing the Rif Mountains and Middle Atlas in Morocco. Then, it filters the Sentinel-2 Surface Reflectance (SR) dataset for images acquired between June 1 and September 30, 2023, within the defined ROI, and with less than 20% cloud cover. The number of images found is printed to the console. Finally, a function `maskS2clouds` is defined to mask clouds using the QA60 band of Sentinel-2, dividing the remaining reflectance values by 10000 to scale them between 0 and 1.

NDWI Calculation and Visualization
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

NDWI Calculation and Visualization
// Apply cloud masking
var maskedDataset = dataset.map(maskS2clouds);

// Function to calculate NDWI
var calculateNDWI = function(image) {
  return image.normalizedDifference(['B3', 'B8']).rename('NDWI');
};

// Calculate NDWI for each image in the collection
var ndwiCollection = maskedDataset.map(calculateNDWI);

// Create a composite image (median NDWI over the time period)
var ndwiComposite = ndwiCollection.median();

// Visualization parameters for NDWI
var ndwiVisParams = {
  min: -1,  
  max: 1,  
  palette: ['red', 'white', 'blue']
};

// Add the NDWI composite to the map
Map.addLayer(ndwiComposite, ndwiVisParams, 'NDWI Composite');

// Center the map on the region of interest
Map.centerObject(roi, 9);
🔑
Beginner note: This code block applies the cloud masking function to the Sentinel-2 dataset. It then defines a function `calculateNDWI` to compute the Normalized Difference Water Index (NDWI) using the green (B3) and NIR (B8) bands. The NDWI is calculated for each image in the collection, and a median NDWI composite is created to represent the typical NDWI value over the specified time period. Finally, the NDWI composite is added to the map with a color palette ranging from red (low NDWI) to blue (high NDWI), and the map is centered on the ROI.

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Export the Chlorophyll-a estimation to Google Drive
Export.image.toDrive({
  image: /* Chlorophyll-a image from DeepSeek result */,
  description: 'Chlorophyll_a_Estimation',
  folder: 'GEE_Exports',
  scale: 10, // Adjust scale as needed
  region: roi,
  maxPixels: 1e13
});

// Export the Turbidity estimation to Google Drive
Export.image.toDrive({
  image: /* Turbidity image from DeepSeek result */,
  description: 'Turbidity_Estimation',
  folder: 'GEE_Exports',
  scale: 10, // Adjust scale as needed
  region: roi,
  maxPixels: 1e13
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.

ElementExample

Date range"filter from 2020-01-01 to 2023-12-31"
Output format"print area in hectares, add a legend, export to Drive"
Error fixing"The code gives error X. Please fix it."
Iteration"Add a time series chart for each year 2018-2023"

Best practice: Start with the main prompt, run the code, then send follow-up prompts to add features one at a time. This gives you cleaner, more reliable code than asking for everything at once.
