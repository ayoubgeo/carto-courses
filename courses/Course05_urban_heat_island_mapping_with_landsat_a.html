Landsat 8 Data Acquisition and NDVI Calculation
10 min

Paste the following code into your GEE code editor and click Run. Read the beginner notes below to understand what each line does.

Landsat 8 Data Acquisition and NDVI Calculation
// Define the region of interest (Fez, Morocco)
var fez = ee.Geometry.Point([ -5.0069, 34.0372]);

// Define the date range
var startDate = '2023-07-01';
var endDate = '2023-08-31';

// Load Landsat 8 SR data
var landsat8 = ee.ImageCollection('LANDSAT/LC08/C01/T1_SR')
  .filterBounds(fez)
  .filterDate(startDate, endDate)
  .filter(ee.Filter.lt('CLOUD_COVER', 20));

// Function to calculate NDVI
var calculateNDVI = function(image) {
  return image.normalizedDifference(['B5', 'B4']).rename('NDVI');
};

// Apply NDVI calculation to the Landsat 8 collection
var ndviCollection = landsat8.map(calculateNDVI);

// Composite the NDVI collection (e.g., median)
var ndvi = ndviCollection.median();

//Display NDVI
Map.addLayer(ndvi, {min: -1, max: 1, palette: ['blue', 'white', 'green']}, 'NDVI');
🔑
Beginner note: This code block first defines the area of interest, the date range, and loads the Landsat 8 Surface Reflectance dataset. It then defines a function to calculate the Normalized Difference Vegetation Index (NDVI) using the Near Infrared (B5) and Red (B4) bands. The NDVI function is applied to the Landsat 8 image collection, and the median NDVI is calculated and displayed.

Land Surface Temperature (LST) Calculation
10 min

Add this block after the previous code. It continues the analysis with additional processing and visualization.

Land Surface Temperature (LST) Calculation
// Function to calculate LST (simplified)
var calculateLST = function(image) {
  var thermalBand = image.select('B10');
  var kelvin = thermalBand.multiply(0.1).add(273.15); // Convert to Kelvin
  return kelvin.rename('LST');
};

// Apply LST calculation to the Landsat 8 collection
var lstCollection = landsat8.map(calculateLST);

// Composite the LST collection (e.g., median)
var lst = lstCollection.median();

//Display LST
Map.addLayer(lst, {min: 290, max: 320, palette: ['blue', 'yellow', 'red']}, 'LST');

// Define a threshold for UHI (e.g., LST > 308 K)
var uhiThreshold = 308;

// Create a UHI mask
var uhiMask = lst.gt(uhiThreshold);

// Display the UHI mask
Map.addLayer(uhiMask.updateMask(uhiMask), {palette: 'red'}, 'UHI Mask');
🔑
Beginner note: This code block calculates Land Surface Temperature (LST) from the Landsat 8 thermal band (B10). It converts the band values to Kelvin. Then, it defines a temperature threshold to identify Urban Heat Island areas. A mask is created based on this threshold and displayed on the map.

Export Results to Google Drive
5 min

Add this block at the end of your script to export the final map as a GeoTIFF to your Google Drive. After clicking Run, go to the Tasks tab in GEE and click Run next to each export task.

Export to Google Drive
// Export the LST image to Google Drive
Export.image.toDrive({
  image: lst,
  description: 'LST_Fez',
  folder: 'GEE_Exports',
  scale: 30,
  region: fez,
  maxPixels: 1e9
});
🔑
Beginner note -- Export.image.toDrive(): This function does not export immediately. It creates a task in the GEE Tasks tab that you must manually start by clicking Run. The file will appear in your Google Drive folder (GEE_Courses) as a GeoTIFF that you can open in QGIS or ArcGIS.
